## 階段のコスト最小化問題　 `stairs_cost.py`
N段の階段を登る時、各段にコストがかかる。コストの列costsが与えられた時に最終段の一個うえ（goal）に到達するのに必要な最小コストを求めるアルゴリズム  
* 階段は、一度に1段もしくは2段登れる
ex) [10, 15, 20] -> 15 (start -> 1(15) -> goal)  

`stairs_cost.py`: dp実装  
`stairs_cost2.py`: 直感的な局所的最適解に陥る実装

## 最長増加部分列 (Longest Increasing Subsequence) `increasing_sub.py`
* 配列の中で、要素が単調増加している最長部分列の長さを求める。
* 連続していなくて良い（順番入れ替えは❌）

ex)  
`[10, 9, 2, 5, 3, 7, 101, 18] -> 4`   
単調増加している最長の部分列は [2, 3, 7, 101] または [2, 5, 7, 18] となります。（要素を飛ばし飛ばし拾ってOK）  
したがって、最大の長さである **4** を返す。
