# 100回建ての建物と2個の卵
最悪何回で、割れる割れないの境界を見つけられるか？


## 考え方
* レベルを区切って考える。区切り目を試していき割れたら最後に割れなかった階の一階上を試す。  
* レベルが上がるごとに、試す回数を使っているので、割れた時の探索に使える回数が1ずつ減る

-> ということは、**連続する整数の和を考えて、100あたりが作れれば良さそう**


1~13の和：　`1 + 2 + ... + 13 = 91`  
1~14の和：　`1 + 2 + ... + 14 = 105`

100下回ると都合がわるい（答えが92~100の時に計算回数がその分増える）ので、初めのレベルのレンジは14から始めた方が良さそう  

はじめに、  
* 14階を試す → 割れたら1, 2, ... 最悪14回

* 27階 (14 + 13)を試す → 最悪は14, 27(割れる), 15, ..., 26の 14回
* 39階 (14 + 13 + 12)を試す → 最悪は14, 27, 39（割れる）, 28, 29, ... ,38 の14回
* ...省略
* 90階を試す　→ 最悪は、14, 27, 39, 50, 60, 69, 77, 84, 90(割れる), 85, 86, 87, 88, 89 の14回
* 95階を試す ->　最悪は 14, ... , 95, 91, 92, 93, 94 の14回
* 99階を試す ->  最悪は 14, ... , 95, 99, 96, 97, 98 の14回
