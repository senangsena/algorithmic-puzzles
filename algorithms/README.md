## 最大の水量を蓄える容器 (Container With Most Water) `water.py`

* 水槽にたくさん壁が立っている
* この中から2つの壁を選んでその間に水を溜めた時、溜められる水の量の最大値は？

**[rule]**   
`water = index_differences × lowest_height(二つの壁のうち低いほう)`   

ex)  
heights = [1, 8, 6, 2, 5, 4, 8, 3, 7] -> 49
