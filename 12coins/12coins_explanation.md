# 🪙 12枚のコイン問題の考え方

「偽物のコイン」と「その重さ」を見つけ出すために、3回しか天秤を使えない。
### そもそもできるの？？
問題が与えられた時に、不可能じゃない？と思うこともありますが、今回は可能であることが証明できる  


考えられる状況は 偽物コインの番号12通り X 軽いか重いか2通り　の、24通り < 天秤3回で表せる状況 3^3 = 27通り  

なので理論的にはできる。  


### 考え方

* ただ、闇雲に乗せても上手くいかないケースが出てくるので、毎回の天秤の使用で、**偏りのない3分岐**を作る必要がある。  
* この問題の全てのつまづきポイントで考えるべきなのはこれだけでOK^^
* つまり、**24通り -> 8通り -> 3通りまたは2通り -> 1通り（特定！**）


## ⚖️ 最初の分割（1回目の天秤）
まずは12枚のコインを **4枚・4枚・4枚** のグループに分け、最初の4枚ずつを比較する。

* **左の皿:** `[1, 2, 3, 4]`
* **右の皿:** `[5, 6, 7, 8]`
* **待機:** `[9, 10, 11, 12]`

### 分岐チェック
この方法で本当に3分岐を有効に使えてる？？  
考えられる状況は全部で24通りなので、24 -> 8通りにすればいいということ。

* つりあったら -> **待機**グループの中にある4通りの番号 X 軽いか重いかの2通り = **8通り**！
* 釣り合わなかったら -> **左の皿**もしくは**右の皿**の中にある8通りの番号 X 1回目の結果(1通り) = **8通り**！

確かに綺麗に均等な3分岐が作れている  

この1回目の天秤の結果によって、大きく2つのルートに分岐する。

---
## 1回目で「つりあった」場合

待機していた `[9, 10, 11, 12]` の中に偽物があると確定

### 【2回目の天秤】の考え方
**前提：天秤一回で使える分岐は3つまで。なので8通り->3通り以下まで絞り込む必要がある**

**ポイント🌟：正常なコインを追加しあえて個数を増やすと綺麗な分岐を作れる**  

左右の皿は同じ枚数だけのせたい。でも4枚だと1枚ずつか2枚ずつになる。2枚ずつでは何も絞り込めないけど1枚ずつだとつりあった時なんの情報もない2枚が残り絞り込み後、`2 × 2 = 4通り > 3通り`でピンチ！そこで、すでに「正常」と確定しているコイン（ここでは1番）を混ぜて5枚で考えてみると、以下のように分けることができて、天秤の結果がなんであれ3通りまで狭めることができる！

* **左の皿:** `[9(疑), 1(正常)]` 
* **右の皿:** `[10(疑), 11(疑)]`
* **除外:** `[12(疑)]`

### 【3回目の天秤】　

* **2回目が「つりあった」場合**
  乗せたものは全て正常。最初に取り除いた **【12番】** が偽物。重いか軽いかのみ特定すればいい ので、

    * **3回目:** `[12]` と `[1(正常)]` を量り、傾きから重いか軽いかを確認する

* **2回目が「傾いた」場合**
  偽物は皿に乗せた `[9, 10, 11]` のどれかです。重い軽いわかっているので、コインの番号のみわかれば良い！　→ 3通りに絞れている

    * **3回目:** 右の皿に乗せていた `[10]` と `[11]` を比べる。この時、**同じ皿から出てきたもの同士比べる**ことに意味がある。例えば`[9]`と`[11]`を比べた時考えられる分岐は、「つりあう」と「2回目と同じ方向に傾く」の**2分岐**のみ。探索を成功させるには毎回3分岐しないといけないのでこれではできない  ...◉

        * **つりあったら:** 残った **【9番】** が偽物。（重い/軽いは2回目の傾きでわかる））

        * **傾いたら:** `[10]` か `[11]` のどちらかが偽物。2回目で「右の皿」側がどう傾いていたか（重かったか、軽かったか）と照らし合わせると特定できる

## 🌪️ ルートB：1回目で「傾いた」場合

偽物の候補は8枚あり、それぞれのコインについてそれが偽物ならば重いのか軽いのかはわかっているので、8通りまで絞られている。ここで偏りのない3分岐を作り出すために、🌟**次の工夫**🌟をする

### 【2回目の天秤】3つのグループへの分割
1. **除外:** 左右から1枚ずつ取り除く
2. **交換:** 左右を1枚ずつ入れ替える。（こうすることで、「1回目と傾きが異なる」という分岐を作れる。）
3. **正解との入れかえ:** このままでは最悪の場合、除外も交換もされていない4枚に絞られ4通りの可能性が残った状態で3回目に進んでしまう。状態は3通り以下にしないといけないので、もう一つ除外し、代わりに正解を入れる

これにより、8枚の候補が **[取り除いた2つ / 交換した2つ / 場所を変えていない3つ]** のグループに偏りなく分割され、以下の配置になります。
* **左の皿:** `[もともと左にいた(残), 右から来た(交), 正常]`
* **右の皿:** `[もともと右にいた(残), もともと右にいた(残), 左から来た(交)]`
* **除外:** `[左から除外, 右から除外, 右から除外]`

### 【3回目の天秤】最終特定
2回目の天秤がどう動いたかで、どのグループに偽物がいるかわかる

* **① 2回目が「つりあった」場合**
  皿の上は全員正解。偽物は、除外した3枚のうちどれか  

  * **3回目:** 右の皿から取り除かれてきた2枚を比べる。なんで同じ皿からのコイン比べるかは、◉の考え方と同じ  
    * **つりあう:** 残りの除外されたコインが偽物。重いか軽いかは1回目の結果からわかっている
    * **傾く:** 2回目の時に元いた皿が重い皿なら、重い方が偽物。逆も然り

* **② 2回目で「傾きが変わった（逆転した）」場合**
  1回目と傾きが逆になった原因は、左右を「交換した」2枚のどちらかです。
    *  **3回目:** 交換した片方と正常を量る
        * **つりあう:** もう片方が偽物！
        * **傾く:** 正常じゃない方が偽物！

* **③ 2回目で「傾きが同じ（変わらない）」場合**
  交換したコインは無関係。ずっと位置を動かさなかった `もともと左にいた(残), もともと右にいた(残), もともと右にいた(残)` のどれか
    * **3回目:** ここからは、◉の考え方と同じ


## 🔑 この問題を解くカギ（面白いところ）

いかに各状況で「天秤が表す3分岐（左・右・つりあい）」を有効に使えるかが、この問題を解く上で大事。

1. **「入れ替え」による新しい分岐の創出** 👀✨
   左右の皿から一つ入れ替えるという発想。こうすることで「傾きが変わる」という新たな状況（分岐）を作ることができる。
2. **「正常なコイン」の活用** ⚖️✨
   このままでは3つの分岐をどう考えても成立できない時に、正解とわかっているコインを一つ加えあえて個数を増やすことで、綺麗に3分岐を表現できる。
